1. What is ReDOS and what part do ‘Evil Regex’ play?

ReDOS uses a malicious regex script to input an infinitely long string of text. This long string causes a server to fail when the string takes so long to parse, performance suffers.

2. What are the common problems associated with the use of regex? How can these be mitigated?

Bugs which allow malicious input. Mitigated by testing.
