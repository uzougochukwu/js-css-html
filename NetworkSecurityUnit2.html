Solar Winds Attack

Main Vulnerabilities

Server Side Request Forgery

In this attack, the adversary modifies the URL in their web browser, in order to gain access to unauthorised information (Al-talak, K. and Abbass, O. 2021).

Cross Site Scripting

Cross Site Scripting involves sending a malicious javascript file to a website, which is then clicked by the victim (Gupta, S. and Gupta, B.B. 2015) .

SQL Injection

The attacker runs a sql script on the web server by typing malicious sql statements into the website’s search engine (Shar, L.K. and Tan, H.B. 2013).
